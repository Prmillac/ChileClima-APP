# App de Clima Chile - MVP
Proyecto para el Módulo 2 de Desarrollo Frontend.

## Características
- **15 ciudades** de Chile integradas dinámicamente.
- **Diseño Responsivo**: Adaptado para móviles (420px) y desktop.
- **Bootstrap 5**: Uso de Cards, Grid, List-groups y Navbar.
- **JS Vanilla**: Manipulación de DOM y gestión de parámetros URL.

## Instalación
1. Descarga el repositorio.
2. Abre `index.html` en cualquier navegador moderno.
